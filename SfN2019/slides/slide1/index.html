Utah arrays: Waveform alterations in recruited vs unrecruited cortex

Example unit (recruited tissue)

Dichotomous activity types in the same seizure

  • Utah array (Blackrock Microsystems) recordings were used as the gold standard for defining ictal recruitment via tonic firing due to their location in layer 4/5 (Schevon et al., 2012)
  • Convex hulls allow waveshape changes equally in any dimension
  • Neurons in recruited tissue showed a population increase in FWHM and decrease in amplitude in a predictable manner (p < 0.001, Mann-Whitney U test; n = 625 units from 6 seizures in 3 patients)
  • In contrast, unrecruited cortical neurons remained stable (5.8% of units p < 0.05 vs. 81.5% in core; Holm-Bonnferroni corrected Mann-Whitney U test; n = 405 units from 4 seizures in 3 patients)
  • This effect was reversible showing return to preictal levels in the postictal period, confirming that this is not an artifact due to patient motion or extrinsic features
  • A Utah array at the edge of recruitment reveals both activity types as ictal propagation reaches the array
